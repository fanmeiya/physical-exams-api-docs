🏥 体检智能分析系统

提供体检报告解读和套餐推荐服务的API

版本 1.0.0

📖 查看API文档 (推荐) 📄 备用文档 ⬇️ 下载规范

🚀 快速开始

# 基础URL
http://localhost:8000

# 健康检查
curl http://localhost:8000/health

📋 主要接口

GET/health

健康检查接口，用于验证服务是否正常运行

POST/health_report

提交健康问卷，支持体检报告解读和个性化推荐

action: "interpret" (体检报告解读) 或 "recommend" (个性化推荐)
GET/task_status/{task_id}

查询任务处理状态和结果

POST/companies/{company_code}/packages/generate

企业管理员生成体检套餐方案

📊 状态说明

PENDING
任务已提交，等待处理
PROCESSING
正在处理中
WAITING
等待进一步操作
DONE
任务完成
ERROR
任务出错

💡 使用示例

curl -X POST http://localhost:8000/health_report \
-H "Content-Type: application/json" \
-d '{
"action": "interpret",
"name": "张三",
"title": "体检报告解读",
"gender": "男",
"age": 35
}'
